# Notebooks

This folder contains Jupyter notebooks for exploratory data analysis, experimentation, and prototyping.

## Naming Convention

Use a numbered prefix to indicate the order and purpose:
- `01_data_exploration.ipynb` - Initial data exploration
- `02_feature_engineering.ipynb` - Feature creation and analysis
- `03_model_development.ipynb` - Model training and evaluation
- `04_results_analysis.ipynb` - Final results and insights

## Best Practices

- Keep notebooks focused on a single topic or analysis
- Move reusable code to modules in `src/`
- Clear outputs before committing (optional but recommended)
- Document assumptions and decisions in markdown cells
